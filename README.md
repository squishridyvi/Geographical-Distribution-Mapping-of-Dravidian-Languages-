This project is about mapping the geographical distribution of Dravidian Languages across India. Work in progress. 

### Current Focus/Interest
-  Kerala (upper and lower district halves) for all identified language populations
-  Uses Census data, PSLI, and GADM shapefiles. Work in progress. 

### Sources: 
- Census of India (2011)
- People's Linguistic Survey of India (PSLI)
- GADM shapefiles (for administrative boundaries)
- Soon: Linguistic Survey of India (LSI) 

### Methods: 
- Cleaning data and restructuring using CSV files taken from census tables
- visualization using Matplotlib and GeoPandas

### Plans: 
- Utilizing the Linguistic Survey of India
- Predicting the geographical spread of different words (lexical diffusion)
